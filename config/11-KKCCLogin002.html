Login con la imagen para el avatar elaborado con HTML y CSS

Hoy les presento como pueden realizar un Loading con HTML y CSS.

   Primero debes crear dos (02) archivos nombrandolos de esta manera index.html, login.css.
En el archivo HTML, se hace el llamado al documento CSS. Después de crear estos archivos, simplemente pegue los siguientes códigos en cada uno de los documentos ya creados.

Archivo HTML


Las siglas HTML quieren decir HyperText Markup Language lo cual significa “lenguajes de marcas de hipertexto”, se basa en un lenguaje de marcas para crear documentos que puedan ser distribuidos por Internet.



Archivo CSS


El CSS (hojas de estilo en cascada) es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado (por ejemplo, HTML). Esta separación entre la estructura y la presentación es muy importante, ya que permite que sólo cambiando los CSS se modifique completamente el aspecto de una página web.

Este archivo realiza un llamado una imagen utilizada como fondo del documento.
